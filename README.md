# Keyboard Test

키보드 버튼을 클릭하거나 실제 키를 눌러 시각적으로 확인하는 키보드 입력 테스트 프로그램입니다.

## 기능

- 마우스 클릭 및 물리 키보드 입력 모두 지원
- 키 입력 시 시각적 하이라이트 효과
- 마지막으로 누른 키 이름 및 코드 표시
- 테스트한 키 개수 카운트
- 초기화 버튼으로 진행도 리셋

## 사용 방법

`keyboard-test.html` 파일을 브라우저에서 열거나, 로컬 서버를 실행하세요.

```bash
python3 -m http.server 8888
```

이후 브라우저에서 `http://localhost:8888/keyboard-test.html` 접속

## 기술 스택

- HTML / CSS / JavaScript (순수 바닐라, 외부 라이브러리 없음)
